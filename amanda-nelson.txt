Questions:
    1. What is Semantic HTML? 
		Semantic HTML are tags that allow us to divide up a website more clearly which can make it more accessible and help with SEO.

    2. What is HTML used for? 		
		HTML is a markup language used for displaying data on a webpage graphically.

    3. What is an attribute and where do we put it? 
		An attribute is a description of where the element is coming from, or what properties to apply to the element. These are inside the opening tag, after the start tag.

    4. What is the h1 tag used for? How many times should I use it on a page?
		The h1 should only be used one time on a webpage and should be used as the main heading.

    5. Name two tags that have required attributes
		img (src- attribute) and a (href-attribute)

    6. What do we put in the head of our HTML document?

    7. What is an id? 
		An id is an attribute that can be given to an element to give it a unique name that be referenced later.

    8. What elements can I add an id to?
		Any element that you want to have a unique name.

    9. How many times can I use the same id on a page? 
		You should only use an id once on a page.

    10. What is a class? 
		A class is an attribute that can be given to an element to give it a name that be referenced later.

    11. What elements can I add a class to? 
		Any element that you want to have groupped together.

    12. How many times can I use the same class on a page? 
		As many times as you'd like, but all these elements with the same class will share the same properties if the class is used later in the page.

    13. How do I get my link to open in a new tab?

    14. What is the alt attribute in the image tag used for? 
		This is used for accessibility readers to read an image description for someone who is not light-abled. This text will also appear if the image source is unable to load on the page.

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
Class and ids are similar in that they give identifiers to elements, but ids can only be used for one single element where classes can be used to classify multiple elements that may share the same attributes. Elements can have multiple classes, and ids, but ids can only be used once per page.
Ex.) <a href=“#” id=”home” class=”navLinks”> Home </a>  “#home only used for the Home link.
<a href=”#” class=”navLinks”> About </a>
<a href=”#” class=”navLinks”> Contact </a> .navLinks for the rest of the navigational links that will have the same characteristics in CSS.

    22. What is a parent child selector? When would this be useful? 
		Parent child selector allows you to call elements that branch from another element. This is super useful to help organize your CSS so other readers can see exactly what element is being called.
			Ex.) HTML
				<div class=”aboutMe”>
					<h3> About <em> Me </em></h3>
			CSS
				.aboutMe h3 em {}
				[parent]   [child, child]
